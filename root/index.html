Transparency

Any changes made to the registry is public

Block-chain is based on the concept of distributed ledgers. Any and all changes made in the block-chain has to be validated by everyone in the network.

Immutable

Data is permanent

Data stored in kyc-chain cannot be modified except through specially written smart contracts which always leave a trail of who accesed them.

Secure

Security like never before

The Data in KYC chain is encrypted using the highly sophisticated SHA-256 encryption. The proof of work needed to modify the block-chain makes it next to impossible to change it

Efficient

Removes redundant KYC verifications

You would no longer need to perform KYC verification for a customer who has been already verified by someone else. The distributed public ledger stores all KYC checks ever performed by all institutions and can be accesssed by any member of kyc-chain

Privacy

Stringent Access Control

The kyc of a person can only be accessed only with his permission. A secret key provided to each person in the block-chain is needed to access or modify his records

Compliance

Adheres to all governmental regulations

You need not worry about being forced to pay fines because of some bank did not do it's KYC checks properly. KYC-chain assigns the risk to the institution which performs the KYC.
